# 🏏 IPL Tracker 2026

A real-time IPL cricket tracking web application.

## Features (planned)
- Live scores
- Upcoming match schedule
- Points table
- Player statistics
- Team profiles
- Stadium info

## Tech Stack
- HTML5 / CSS3 / Vanilla JavaScript
- Cricket API (cricapi.com)
- Google Fonts (Rajdhani + DM Sans)

## Setup
```bash
git clone https://github.com/yourname/ipl-tracker.git
cd ipl-tracker
open index.html
```
